## 重学JavaScript|编程语言通识与JavaScript语言设

### 语言按语法分类
1. 非形式语言
    * 中文
    * 英文
2. 形式语言（乔姆斯基谱系）
    * 0型 无限制文法
    * 1型 上下文相关文法
    * 2型 上下文无关文法
    * 3型 正则文法

1. Atom

2. Express
    * Atom
    * Operator
    * Punctuator
3. Statement
    * Expression
    * KeyWord
    * Punctuator
4. Structure
    * Function
    * Class
    * Process
    * Namespace
    * ......
5. Program
    * Program
    * Module
